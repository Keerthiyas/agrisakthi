# AgriSakthi

AI-Powered Agriculture Assistant for Smart Farming

AgriSakthi is an AI-powered agriculture assistance platform designed to help farmers make informed decisions using Artificial Intelligence, Machine Learning, and Natural Language Processing.

The platform provides intelligent support for crop selection, plant disease detection, weather updates, and government scheme information through a simple and interactive interface.

This project aims to empower farmers with technology and improve agricultural productivity.

## 📌 Problem Statement

Farmers often face challenges such as:
- Lack of knowledge about suitable crops for their land
- Difficulty in identifying plant diseases
- Limited access to weather information
- Unawareness of government agricultural schemes

AgriSakthi solves these problems by providing AI-driven guidance and recommendations.

## 🚀 Features

### 🤖 AI Chatbot
An intelligent chatbot that answers farmer queries related to:
- Crop recommendations
- Fertilizers and nutrients
- Pest control
- Farming practices

Farmers can interact with the chatbot in a simple conversational way.

### 🌱 Crop Recommendation System
The system suggests suitable crops based on parameters like:
- Soil type
- Temperature
- Humidity
- Rainfall

This helps farmers choose crops that grow better in their environmental conditions.

### 🦠 Plant Disease Detection
Farmers can upload an image of a plant leaf and the system will:
- Detect the disease using image classification
- Display the disease name
- Suggest prevention and treatment methods

This feature uses machine learning and computer vision.

### 🌦 Weather Updates
The platform provides weather information such as:
- Temperature
- Humidity
- Rainfall prediction

This helps farmers plan irrigation, fertilization, and harvesting.

### 🏛 Government Scheme Information
The system displays information about:
- Agricultural subsidies
- Crop insurance
- Farmer welfare programs

This helps farmers access government support easily.
